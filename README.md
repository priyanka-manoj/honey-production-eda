# Honey Production Analysis (EDA)

## Overview

This project explores the US honey production dataset to understand trends in:

* Honey production over time
* Yield per colony
* Number of colonies
* Price and production value
* Impact of Colony Collapse Disorder (CCD)

## Goal

The goal of this notebook is to practice exploratory data analysis and understand:

* What changed in honey production over time
* Why those changes happened
* How different states behaved differently

## Key Focus Areas

* Overall production trends (1998–2016)
* State-wise differences in production
* Relationship between yield, colony count, and total production
* Impact of CCD on productivity and supply
* Price trends and market response

## Key Insights

* Yield per colony has declined over time, especially after 2006
* Production trends vary by state
* Some states (e.g., North Dakota) increased production by increasing colony count
* Other states (e.g., California) saw declines in both yield and colonies
* Price per pound increased over time, suggesting supply pressure
* Production is becoming more scale-driven than efficiency-driven
